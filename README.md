- 元データ: Canva テンプレート「ブラウン　ピンク　シンプル　ミニマリスト　結婚　両家顔合わせ　しおり　A4文書」書き出し PNG（表紙・見開き）
- 切り出し画像: テキスト版のプロフィール・家族紹介用に、`spread_inner.png` から `profile_*.png` / `fam_*.png` を生成（Canva の見開きを差し替えた場合は座標の見直しが必要）
